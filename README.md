# 🤖 OHOOO BOT - Telegram Group Manager

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.9+](https://img.shields.io/badge/Python-3.9%2B-blue.svg)](https://www.python.org/)
[![Telegram Bot API](https://img.shields.io/badge/Telegram-Bot%20API-blue.svg)](https://core.telegram.org/bots/api)

Bot Telegram multifungsi untuk manajemen grup dengan fitur lengkap, game, dan AI integration.

## ✨ Fitur Utama

### 🛡️ Admin Tools
- Anti-spam protection
- Welcome messages
- Auto-kick/ban
- Word filter
- Mute/unmute users
- Warn system
- Message pinning

### 👥 User Features
- Profile info
- Group stats
- Rank system
- AFK mode
- Reminders

### 🎮 Games & Fun
- Truth or Dare
- Tebak kata
- Math quiz
- Random memes
- Quotes generator

### 🤖 AI Features
- ChatGPT integration
- Image generation
- Translation
- Text-to-speech

### 📊 Analytics
- Group statistics
- User activity tracking
- Message counting

## 🚀 Quick Start

### Prerequisites
- Python 3.9+
- Telegram Bot Token from [@BotFather](https://t.me/BotFather)
- API keys (optional for AI features)
